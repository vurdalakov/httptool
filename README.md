# httptool

#### Overview

**httptool** is a simple command-line tool that executes HTTP methods.

This tool is part of [`dostools` collection](https://github.com/vurdalakov/dostools). 

Distributed under the [MIT license](http://opensource.org/licenses/MIT).

#### Command line syntax

```
httptool <-method> <url> [-options]
```

##### HEAD method

```
httptool head <url> [-options]
```

Examples:

```
httptool head http://www.booksite.ru/krassever/1917/1917_1.pdf
```
